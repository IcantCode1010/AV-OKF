# AV-OKF Web

Stage 0 product shell for the AV-OKF document intelligence platform.

## Commands

```bash
pnpm --dir apps/web dev
pnpm --dir apps/web lint
pnpm --dir apps/web build
```

## Current Scope

- Mock auth and workspace context
- Seeded demo documents
- Dashboard shell
- Document library shell
- Document detail shell
- Settings shell

Stage 1 adds real PDF upload, file storage, editable metadata, tags, and processing state.
